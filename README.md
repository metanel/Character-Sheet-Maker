# 🎨 יוצר דף דמות מקצועי - Character Sheet Maker

אפליקציית אינטרנט חינמית ליצירת דפי דמות (Character Sheets) מקצועיים לאנימציה, משחקים, ומודלינג תלת-ממדי.

## ✨ תכונות

- 🎭 **3 סוגי דפי דמות:**
  - פנים בלבד (6 זוויות כולל זוויות אחוריות)
  - גוף בלבד (4 זוויות)
  - משולב פנים + גוף (7 תמונות)

- 🎨 **עיצוב צבעוני וידידותי**
- 📸 **הנחיות חזותיות מפורטות לכל זווית**
- 🖼️ **יצוא באיכות הדפסה גבוהה (300 DPI)**
- 🔒 **פרטיות מלאה** - כל העיבוד נעשה בדפדפן שלך
- 💾 **הורדה והדפסה ישירה**
- 📱 **תמיכה מלאה במובייל וטאבלט**

## 🚀 שימוש מהיר

## 📖 הנחיות שימוש

### 1️⃣ בחר סוג דף דמות

- **פנים בלבד**: למודלים, שחקנים, אנימציה פנים
- **גוף בלבד**: עיצוב אופנה, משחקי וידאו
- **פנים + גוף**: דף דמות מלא ומקצועי

### 2️⃣ צלם את התמונות

#### טיפים לצילום מקצועי:

✅ **רקע:**
- השתמש ברקע אחיד (לבן, אפור, או כחול בהיר)
- הימנע מצללים וממרקמים

✅ **תאורה:**
- תאורה רכה ואחידה
- הימנע מצללים חזקים על הפנים/גוף
- תאורה טבעית או softbox מומלצים

✅ **מצלמה:**
- שמור על גובה ומרחק אחידים בכל התמונות
- השתמש בחצובה לייצוב
- רזולוציה גבוהה (לפחות 1920x1080)

✅ **תנוחה:**
- תנוחת A טבעית לגוף (רגליים מעט פתוחות)
- ביטוי פנים נייטרלי
- שיער מסודר וגלוי

### 3️⃣ העלה והורד

1. העלה תמונה לכל חריץ (Drag & Drop או לחיצה)
2. אשר שכל התמונות תקינות
3. לחץ "צור קולאז'"
4. הורד או הדפס את התוצאה

## 🎯 למי זה מתאים?

- 🎬 **אנימטורים** - Reference sheets לאנימציה תלת-ממדית
- 🎮 **מעצבי משחקים** - Character models למשחקים
- 👗 **מעצבי אופנה** - תיק עבודות למודלים
- 🎭 **שחקנים** - Casting sheets מקצועיים
- 🖌️ **אמני 3D** - Reference לפיסול ומודלינג
- 📸 **צלמים** - תיעוד מקצועי

## 🛠️ טכנולוגיות

- HTML5 Canvas (לעיבוד תמונות)
- JavaScript Vanilla (ללא תלויות חיצוניות)
- CSS3 (עיצוב רספונסיבי)
- 100% Client-Side (אין שרתים)

## 🔒 פרטיות ואבטחה

- ✅ התמונות **לא נשלחות לשרת**
- ✅ אין שמירת מידע או עוקבנים
